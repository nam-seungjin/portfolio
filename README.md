# portfolio
## 목적
내가 직접 제작하고자 시안디자인을 했고 그 디자인을 통해 웹 포트폴리오를 제작하였습니다.
이 포트폴리오 제작을 통해 저에 대해서 더 알 수 있도록 제가 할 수 있는 기술들을 보여주는 것과 제가 직접 한 작업들
그리고 스크롤되는 이벤트도 만들어서 제작하였습니다.
### 개선
평소 웹사이트를 보면서 내가 했던 작업이나 디자인을 보여주기 위해 사진을 클릭했을 시에 팝업창으로 내가 했던 것을 디자인해서 보여주는 형식으로
제작했습니다. 그리고 스크롤 이벤트를 주면서 네비항목을 클릭했을때도 TOP버튼을 눌렀을때에도 스크롤 될 수 있도록 하였습니다.
